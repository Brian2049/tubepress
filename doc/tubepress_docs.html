Put the following in your posts/pages to make your gallery show up. Note that you can specify different options on different pages. Get creative!


[triggerword {options}]


triggerword is defined in the main TubePress options page. Default is 'tubepress'


{options} is an optional combination of the following comma-separated name-value pairs.

Options that occur on the same line must be specified together. The default values are given as examples.

You can optionally put single or double quotes around option values, but if the value consists of

more than one word, you *must* put quotes around it; mode=tag, tagValue='spongebob squarepants'


mode=favorites

Selects which videos to display for this gallery.

mode=favorites, favoritesValue=3hough

Videos from the specified user's "favorites" list

mode=tag, tagValue=(space-separated tags with single quotes around them)

Videos matching all of the specified tags. Use quotes for more than one tag.

mode=related, relatedValue=(space-separated tags with single quotes around them)

Videos matching any of the specified tags. Use quotes for more than one tag.

mode=user, userValue=someuser

Videos from the specified user

mode=playlist, playlistValue=someplaylistid

Videos from the specified playlist

mode=popular, popularValue=(valid time frame)

Most-viewed videos on YouTube during specified time frame.

Valid time frames are: 'day', 'week', 'month'

mode=featured

The latest 25 videos featured on YouTube

playerLocation=normal

Selects where the videos will play

playerLocation=normal

Plays videos at the top of the gallery

playerLocation=new_window

Plays by themselves in a new page (non-popup)

playerLocation=popup

Plays videos in a popup window

playerLocation=thickbox

Plays videos using ThickBox (experimental)

playerLocation=youtube

Plays videos from the original YouTube page


mainVidHeight=325

Max height (in px) of "main" video. Valid for all 

'playerLocation' options except for 'youtube'


mainVidWidth=450

Max width (in px) of "main" video. Valid for all

  'playerLocation' options except for 'youtube'

thumbHeight=90

Max height (in px) of thumbnails


thumbWidth=120

Max width (in px) of thumbnails


(meta value)=(true/false)

Toggles display of video meta information in the gallery.

Where valid meta values are: title, length, author, views, id,

rating, ratings, uploaded, comments, url, thumburl, tags, description


timeout=6

How long to wait (in seconds) for YouTube to respond before timing out


resultsPerPage=15

How many videos to display per page. Only valid for the following

modes: 'user', 'tag', 'youtube'


As of version 1.05, ThickBox (http://jquery.com/demo/thickbox) is disabled by default. It's interfering with themes, Lightbox, and other jquery-enabled features for Wordpress. Until I come up with a fix, you'll have to manually enable it. Edit /wp-content/plugins/tubepress/tubepress.php and change line 39 from:


$tubepress_enable_thickbox = false;


to


$tubepress_enable_thickbox = true;
